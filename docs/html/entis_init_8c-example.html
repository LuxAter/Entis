Entis
entis_init.c

Initialize Entis library.This must be called before all other Entis calls. It generates the XCB connection to the X server, which is necessary for almost all functions present in the Entis library. Thus be sure to call this function before any other functions from Entis.

Parameters
titleTitle of the window to generate.
wDesired width of the window.
hDesired height of the window.
value_maskBit mask of values to alter the window.
value_listValues associated with provided bits in value_mask.
entis_init("Entis Window", 500, 500, 0, NULL);
